To enable real Stripe payments:

  1. Get your Stripe Secret Key:
    • Go to Stripe Dashboard ‚Üí API Keys
    • Copy your Secret key (starts with sk_live_)
  2. Create a Product & Price:
    • Go to Stripe Dashboard ‚Üí Products
    • Create a new product called "BizPal Pro"
    • Add a recurring price of 99 SEK/month
    • Copy the Price ID (starts with price_)
  3. Update the server configuration:
    • Create a .env file in the project root
    • Add: STRIPE_SECRET_KEY=sk_live_your_secret_key_here
    • Add: STRIPE_PRO_PRICE_ID=price_your_price_id_here
  4. Restart the server:
    • Stop the current server (Ctrl+C)
    • Run: node server.js
